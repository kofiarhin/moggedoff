# Moggoff Project Brief

## 1. Project Overview

Moggoff is a fun AI-powered web app where users upload two selfies and
the system decides which one "mogged" the other.

## 2. Project Goal

Build an MVP that compares two selfies using AI and returns a winner
instantly.

## 3. Core Flow

Upload → Analyze → Compare → Show Winner

## 4. Tech Stack

Frontend: React + Vite + Tailwind\
Backend: Node.js + Express\
AI: Azure Face API

## 5. API

POST /api/battles/analyze

## 6. Env

AZURE_FACE_API_KEY=your_key AZURE_FACE_API_ENDPOINT=your_endpoint

## 7. MVP Success

-   Upload works
-   AI returns result
-   Winner displayed
-   Files cleaned
